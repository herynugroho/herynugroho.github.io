⚠️ Kondisi Saat Ini

Alur Operasional Existing (Sebelum Integrasi API Bank)

Proses bisnis sangat bergantung pada intervensi manusia (human intervention). Menyebabkan perlambatan (bottleneck) hebat terutama pada awal bulan saat tagihan membeludak.

FASE 1 ✉️

Penagihan Manual

Sistem PINISI membuat invoice, lalu admin mengirimkan pesan tagihan ke pelanggan berisi nominal dan nomor rekening umum perusahaan secara manual lewat WhatsApp.

Langkah Lambat
FASE 2 💸

Titik Kritis Pembayaran

Pelanggan transfer dana, namun wajib mengirimkan bukti transfer ke WA Admin/CS. Jika lupa mengirim, status pembayaran mengambang (floating).

Risiko Tinggi
FASE 3 🔍

Verifikasi Manual

Tim Finance login portal perbankan, mengecek mutasi satu per satu, mencocokkan nominal dan nama. Sangat rawan human error jika nama rekening pengirim berbeda.

Sangat Rawan Error
FASE 4 ⚙️

Aktivasi Manual

Setelah divalidasi, Admin mengubah status di PINISI menjadi "Lunas", lalu secara terpisah memicu perintah pembukaan isolir pelanggan di dashboard ACS/Mikrotik.

Keterlambatan Layanan
💡

Kesimpulan Bottleneck: Hubungan manual dari bank ke internet aktif memerlukan waktu berkisar antara 15 menit hingga beberapa jam, bergantung penuh pada ketersediaan admin di jam kerja.

⚡ Target Solusi Integrasi

Alur Operasional Rencana (Mandiri Host-to-Host)

Dengan sistem Mandiri Bill Collection, intervensi manusia dipangkas habis. Sistem PINISI dan Bank Mandiri akan "berbicara" satu sama lain secara otomatis.

1. TAGIHAN & VA

Kalkulasi & Request VA

Sistem PINISI mengkalkulasi tagihan, mengirim request ke Mandiri secara H2H, dan mengunci nominal dengan mekanisme Close Payment. Mandiri merespons dengan nomor unik Virtual Account (VA).

2. PEMBAYARAN

Multi Channel & Instan

Pelanggan membayar lewat ATM, mobile banking bank manapun, atau e-wallet dengan nomor VA yang diterima. Pembayaran langsung masuk ke rekening perusahaan.

3. CALLBACK REAL-TIME

Deteksi Pembayaran

Begitu transaksi berhasil, Bank Mandiri langsung mengirim notifikasi real-time ke server PINISI. Invoice otomatis terdeteksi Lunas tanpa campur tangan admin.

4. AUTO-AKTIVASI

Penormalan Instan

Karena invoice lunas otomatis, sistem PINISI langsung memicu script API ke server ACS/jaringan Anda untuk menormalkan kembali koneksi internet pelanggan dalam hitungan detik.

5. REKONSILIASI

Mitigasi Keuangan

Bank Mandiri melakukan rekonsiliasi pada H+1 dengan mengirimkan berkas laporan rekening koran berformat MT940 harian via File Transfer/email secara terjadwal.

✓

Hasil Akhir: Seluruh proses pembayaran hingga aktivasi internet selesai dalam waktu kurang dari 10 detik, berjalan otomatis 24/7, tanpa membutuhkan verifikasi admin sama sekali.
