⚠️ Kondisi Saat Ini

Alur Operasional Existing (Sebelum Integrasi)

Proses bisnis sangat bergantung pada intervensi manusia (*human intervention*), menyebabkan perlambatan hebat saat awal bulan tagihan.

FASE 1

Penagihan Manual

Sistem PINISI membuat invoice, lalu admin mengirimkan pesan tagihan ke pelanggan secara manual via WhatsApp.

Langkah Lambat
FASE 2

Titik Kritis Bayar

Pelanggan wajib mengirimkan foto bukti transfer ke WA Admin. Jika lupa, status pembayaran mengambang.

Risiko Tinggi
FASE 3

Verifikasi Manual

Tim Finance login portal perbankan, mengecek mutasi satu per satu, mencocokkan nominal. Sangat rawan *human error*.

Rawan Error
FASE 4

Aktivasi Manual

Admin mengubah status di PINISI menjadi "Lunas", lalu memicu perintah buka isolir pelanggan di dashboard ACS/Mikrotik.

Delay Layanan

Kesimpulan Bottleneck: Hubungan manual dari bank ke internet aktif memerlukan waktu berkisar antara 15 menit hingga beberapa jam, bergantung penuh pada ketersediaan admin di jam kerja.

⚡ Target Solusi Integrasi

Alur Operasional Masa Depan (Mandiri H2H)

Dengan sistem Mandiri Bill Collection, intervensi manusia dipangkas habis. Sistem PINISI dan Bank Mandiri berkomunikasi otomatis.

1. INVOICING

Request VA

PINISI meminta nomor Virtual Account ke Mandiri dengan mekanisme *Close Payment* (nominal dikunci).

2. TRANSACTION

Multi Channel

Pelanggan membayar lewat ATM, mobile banking bank manapun, atau e-wallet secara instan.

3. CALLBACK

Real-Time

Bank Mandiri langsung mengirim notifikasi sukses transaksi ke server PINISI secara detik itu juga.

4. SYSTEM WORK

Auto-Aktivasi

Sistem PINISI otomatis memicu script API ke server ACS untuk membuka isolir internet pelanggan.

5. AUDIT

Rekonsiliasi

Bank melakukan sinkronisasi berkas laporan rekening koran format MT940 harian otomatis via File Transfer.

Hasil Akhir: Seluruh proses pembayaran hingga aktivasi internet selesai dalam waktu kurang dari 10 detik, berjalan otomatis 24/7, tanpa membutuhkan verifikasi admin sama sekali.
