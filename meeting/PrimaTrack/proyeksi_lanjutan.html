Fase Pasca-Inkubasi (Bulan ke-7 dts)

Proyeksi Pendapatan Berulang & Ekspansi

Basis Armada: 150 Unit (Telah Berstatus Aset Kampus)

Potensi 1: AMC (Annual Maintenance Contract)

Kontrak perpanjangan otomatis yang wajib dibayar kampus agar 150 unit GPS tetap mendapatkan koneksi internet dan akses ke Server PrimaTrack selama 1 Tahun ke depan (12 Bulan). Kami asumsikan biaya perpanjangan yang ditawarkan sangat kompetitif, yaitu Rp 1.500.000 / unit / tahun (setara Rp 125.000/bulan).

Nilai Kontrak Tahunan

Rp 225.000.000

150 unit x Rp 1.500.000

Total Modal (Data + Server + OPEX)

Rp 57.000.000

Hanya biaya maintenance murni

Laba Bersih Tahunan

Rp 168.000.000

Margin: 74.6%

Rincian Modal (Beban Tahunan):

  • Kuota IoT Telkomsel (150 unit x 12 bln x Rp 15.000) Rp 27.000.000
  • Sewa Private Server VPS Tahunan Rp 15.000.000
  • Biaya Kunjungan Teknisi Lapangan (Buffer) Rp 15.000.000

Potensi 2: Ekosistem Proyek Lanjutan (Tahun ke-2 dts)

Karena server dan API data sudah Bapak kuasai, Bapak memegang monopoli akses atas data armada kampus. Ini membuka peluang SPK (Surat Perintah Kerja) baru untuk pengembangan perangkat lunak kampus:

Aplikasi Pelacakan Bus Mahasiswa

Nilai: Rp 75 - 150 Jt

Pembuatan aplikasi mobile (Android/iOS) berbasis API PrimaTrack agar mahasiswa dapat melihat posisi bus kampus secara langsung. Sangat mendukung penilaian Smart Campus untuk akreditasi BAN-PT.

Integrasi HRD & Absensi RFID

Nilai: Rp 50 - 100 Jt

Proyek integrasi sistem untuk mencocokkan data titik berhenti GPS dengan jadwal tugas sopir kampus, sehingga HRD dapat otomatis menghitung uang makan, uang jalan, dan overtime sopir secara presisi.

Penambahan Armada Baru

Recurring Hardware

Setiap ada pengadaan unit kendaraan dinas baru dari Rektorat (misal tahun depan ada 10 mobil baru), pihak kampus wajib membeli GPS dari perusahaan Bapak agar terhubung ke dashboard yang sama (Vendor Lock-in).

Total Potensi Nilai Proyek (Tahun Ke-2)

Dengan menyetujui proyek awal (Rp 500 Juta / 6 Bulan), Bapak secara otomatis membuka keran pendapatan di tahun berikutnya dengan total potensi nilai mencapai Rp 350.000.000 hingga Rp 475.000.000 (Gabungan AMC dan Proyek Turunan), dengan margin laba bersih jauh di atas 70% karena infrastruktur utama telah terbangun.
